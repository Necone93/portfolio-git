eduZone - admin panel

The admin panel allows teachers to manage students by assigning them to specific classes and grades. Behind the scenes, a structured database supports storing tests, lessons, homework, and announcements. Teachers can assign tests during class in real-time, and once students complete them, results are automatically sent to the teacher’s email. Additionally, educators can upload learning materials and communicate important updates directly through the platform.

Interactive Landing Page – Responsive Web Design Project

A visually engaging and fully responsive landing page built using only HTML and CSS, created as part of my practice while learning advanced layout and animation techniques. The site features a custom sidebar menu with a hamburger toggle, animated banner text, interactive service cards, and responsive layout using both CSS Grid and Flexbox. Media queries are used to ensure optimal display across desktop and mobile devices.
Technologies used: HTML5, CSS3 (Flexbox, Grid, Keyframe Animations, Media Queries), Git & GitHub, GitHub Pages (for deployment)
